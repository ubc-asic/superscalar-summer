## [RV32E](https://docs.riscv.org/reference/isa/unpriv/unpriv-index.html)

Reduces number of integer registers to 16 general-purpose registers (x0-x15).
Each register is 32 bits wide.

| Register | Description |
| -------- | ----------- |
| x0  | dedicated zero |
| x1  | return address |
| x2  | stack pointer  |
| x3  |   |
| x4  |   |
| x5  | alternate link register |
| x6  |   |
| x7  |   |
| x8  |   |
| x9  |   |
| x10 |  |
| x11 |  |
| x12 |  |
| x13 |  |
| x14 |  |
| x15 |  |
| pc | program counter - address of current instruction |
